## 🚀 Быстрый старт (5 минут)

### 1️⃣ Установка зависимостей

**macOS/Linux:**
```bash
cd google-ads-project
chmod +x setup.sh
./setup.sh
```

**Windows:**
```bash
cd google-ads-project
setup.bat
```

### 2️⃣ Получить Telegram Bot Token

1. В Telegram найти `@BotFather`
2. Отправить `/newbot`
3. Следовать инструкциям
4. Скопировать токен в `.env`

### 3️⃣ Получить Chat ID

1. Открыть своего бота
2. Отправить `/start`
3. Перейти: `https://api.telegram.org/bot<TOKEN>/getUpdates`
4. Найти `"id"` - это ваш Chat ID
5. Скопировать в `.env`

### 4️⃣ Запустить локально

**Терминал 1 (Backend):**
```bash
source venv/bin/activate      # macOS/Linux
# или: venv\Scripts\activate.bat  # Windows
python api/app.py
```

**Терминал 2 (Frontend):**
```bash
cd google-ads-project
python -m http.server 8000
```

**Открыть:** http://localhost:8000

### 5️⃣ Тестирование

- Заполнить форму калькулятора
- Отправить заявку
- Проверить Telegram - вы получите сообщение! ✅

**1. Heroku (Самый простой)** ⭐
```bash

heroku config:set TELEGRAM_BOT_TOKEN=xxx
```
