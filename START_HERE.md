Затем откройте в браузере:
```
http://localhost:8000
```

### Способ 2: Полный (С Telegram интеграцией)

**Шаг 1: Получить Telegram Bot Token**

1. В Telegram найти `@BotFather`
2. Написать `/newbot`
3. Следовать инструкциям
4. Скопировать токен

**Шаг 2: Получить Chat ID**

1. Написать своему боту `/start`
2. Перейти: `https://api.telegram.org/bot<СКОПИРОВАТЬ_ТОКЕН>/getUpdates`
3. Найти `"chat"` -> `"id"` (это ваш Chat ID)

**Шаг 3: Настроить .env**

Откройте файл `.env` и заполните:
```env
TELEGRAM_BOT_TOKEN=скопировать_сюда_токен
TELEGRAM_CHAT_ID=скопировать_сюда_chat_id
WHATSAPP_PHONE=ВАШ_НОМЕР_БЕЗ_СКОБОК
```

**Шаг 4: Установить Python зависимости**

```bash
cd /Users/igorabovyan/Desktop/google-ads-project

# Создать виртуальное окружение
python3 -m venv venv

# Активировать
source venv/bin/activate  # macOS/Linux
# или: venv\Scripts\activate.bat  # Windows

# Установить зависимости
pip install -r requirements.txt
```

**Шаг 5: Запустить Backend (Telegram интеграция)**

```bash
python api/app.py
```

Вы увидите:
```
WARNING in app.run_simple...
Running on http://127.0.0.1:5000
```

**Шаг 6: Запустить Frontend (В новом терминале)**

```bash
cd /Users/igorabovyan/Desktop/google-ads-project
python3 -m http.server 8000
```

**Шаг 7: Открыть в браузере**

```
http://localhost:8000
```

**Шаг 8: Тестировать**

1. Заполнить форму калькулятора
2. Нажать "Get My Offer Now"
3. Проверить Telegram - вы получите сообщение! ✅

## ❓ ЧАСТЫЕ ВОПРОСЫ

**Q: Как получить Telegram токен?**  
A: Напишите `@BotFather` в Telegram, отправьте `/newbot`, и он выдаст вам токен.

**Q: Почему форма не отправляется в Telegram?**  
A: Проверьте что .env заполнен правильно (без пробелов, без кавычек).

**Q: Можно ли менять цвета на сайте?**  
A: Да, в файле `css/styles.css` в переменных `:root` (строка ~8-20).

**Q: Как добавить новый город?**  
A: Скопируйте файл `pages/toronto.html`, переименуйте, отредактируйте текст.

**Q: Можно ли менять алгоритм калькулятора?**  
A: Да, в файле `js/calculator.js` переменные `carPricingData` и `carBasePrices`.
